# antonio-claude-workspace

Repositorio de trabajo colaborativo entre **Antonio** y **Claude Code**.

## Estructura

```
antonio-claude-workspace/
├── projects/        # Proyectos en desarrollo
├── experiments/     # Prototipos y pruebas rápidas
├── scripts/         # Scripts de utilidad reutilizables
├── docs/            # Documentación y notas técnicas
└── CONTEXT.md       # Contexto compartido y decisiones de diseño
```

## Cómo trabajamos

- **Antonio** define objetivos, revisa resultados y toma decisiones de producto.
- **Claude Code** implementa, refactoriza, depura y documenta.
- Cada sesión de trabajo parte del estado actual del repo — los cambios se commitean con mensajes claros.
- Las decisiones importantes y el contexto acumulado viven en `CONTEXT.md`.

## Convenciones de commits

```
feat:  nueva funcionalidad
fix:   corrección de bug
docs:  cambios de documentación
chore: mantenimiento, scripts, configuración
refactor: mejora interna sin cambio de comportamiento
```

## Inicio rápido

```bash
git clone https://github.com/antoniobenavides-py/antonio-claude-workspace.git
cd antonio-claude-workspace
```
