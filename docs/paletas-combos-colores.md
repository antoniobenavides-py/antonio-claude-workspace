## COMBO 01 — Tiffany + Dark Gray

| Rol visual | Nombre | HEX |
|---|---|---|
| Principal / luminoso | Tiffany | `#21F1A8` |
| Base / oscuro | Dark Gray | `#171717` |

**Sensación visual:** tecnológica, fresca, moderna y de alto contraste.
**Uso recomendado:** interfaces dark mode, landing pages de IA, dashboards, CTAs energéticos.
**Asignado a:** vibecodinges.com (principal)

```css
:root {
  --combo-01-tiffany:   #21F1A8;
  --combo-01-dark-gray: #171717;
}
```
